🛍️ Retailmart Analytics Dashboard
📊 Project Overview

The Retailmart Analytics Dashboard is an end-to-end data analysis project designed to extract meaningful insights from retail sales data.
It focuses on understanding sales performance, customer behavior, and product trends through interactive visualizations.

🎯 Objectives
Analyze overall sales performance
Identify top-performing products and categories
Understand customer purchasing behavior
Track regional and monthly sales trends
🛠️ Tools & Technologies
SQL (PostgreSQL) – Data extraction & querying
Power BI / Excel – Data visualization (update based on your tool)
GitHub – Project hosting & version control
📂 Dataset Information

The dataset contains retail transaction data, including:

Orders
Customers
Products
Sales, Quantity, and Revenue
